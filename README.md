# Car Dealership

- Aplicación creada en el Curso de Nest.js de Fernando Herrera

## Deploy en Local

- Instalamos dependencias

```
  npm install
```

- Corremos app

```
  npm run dev
```

- Corremos nuestro seed para obtener datos estáticos de nuestros endpoints

```
  localhost:3000/seed
```

## Endpoints

- Listado de Cars

```
localhost:3000/cars
```

- Listado de brands

```
localhost:3000/brands
```
